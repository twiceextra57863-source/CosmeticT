# TButtonMod

A modern Fabric mod that adds a convenient 'T' button to the Minecraft Title Screen.

## Features
- **Dashboard Screen**: Centralized place for mod settings.
- **Theme Settings**: Customizable modern UI elements.
- **Skin Preview**: Inspect player skins directly.
- **Modern UI Widgets**: Provides a clean look to custom GUIs.

## Building
Run `./gradlew build` to compile the mod.

## Compatibility
Built for Fabric Minecraft 1.20.4.
